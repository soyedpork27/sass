SASS - SASS(Syntactically Awesome StyleSheets) 소개

CSS pre-precessor(전처리기)로서 css 반복을 줄여 시간을 절약할 수 있도록 해주는 css 언어의 확장 언어이다.

변수 및 조건문 등의 새로운 문법을 통해 css를 구조화하여 작업할 수 있다.

코드가 지저분해질 수 있고, 코드가 많아질수록 유지 보수가 어렵다.

프로그램 설치는 sass 공식 사이트에서 gui, cli 방식의 프로그램들 중 설치한다. vs code 플러그인 'sass live compiler'를 설치하면 좀 더 편하게 사용 가능하다.

sass 파일의 확장자는 .sass , .scss 2 가지가 존재하며 문법의 차이가 있고 거의 같다고 보면 된다.

css 문법과 거의 비슷하며 주로 scss파일을 사용하며, 브라우저에서는 css만 인식하기 때문에 .scss파일을 .css 로 변환하는 작업을 거쳐야 사용할 수 있다.
